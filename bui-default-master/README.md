bui-default
===========

基于BUI控件库的一个应用
- [文档地址](http://www.builive.com) 

## 使用的控件和样式

- 这个应用是基于[BUI控件库](https://github.com/dxq613/bui)
- 所使用的样式基于[DPL](http://www.builive.com/base/index.php)
